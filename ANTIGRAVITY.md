# Antigravity Instructions

Use this project as a local research-and-extraction kit.

Primary objective:

Find reliable open classical or scholarly texts, build a source manifest, verify rights status, extract clean text, create citation data, and prepare Korean translation output.

Secondary objective:

When the user starts with a Korean translation or remembered sentence, run the multilingual reverse-trace flow and produce a candidate report rather than a single asserted answer.

Start with:

- `README.md`
- `AGENTS.md`
- `instructions/COMMON_WORKFLOW.md`
- `instructions/SOURCE_POLICY.md`
- `instructions/REVERSE_TRACE_MULTILINGUAL.md`

Run scripts from the project root. Do not write generated files outside `work/` or `results/`.
