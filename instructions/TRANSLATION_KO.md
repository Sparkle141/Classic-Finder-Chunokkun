# Korean Translation Guidance

## Output Shape

Use a source-aligned format:

```markdown
## Passage 001

Source:
...

Korean:
...

Notes:
...
```

## Style

- Translate accurately before polishing.
- Keep key philosophical, legal, religious, and philological terms consistent.
- Use modern Korean syntax unless the user asks for archaic Korean.
- Preserve named entities, section titles, and technical terms.
- Add translator notes only when they clarify a real interpretive issue.

## Term Table

Maintain a table when recurring terms matter:

```markdown
| Source Term | Korean Term | Note |
|---|---|---|
| commonwealth | 국가 / 코먼웰스 | Choose one and keep it consistent. |
| covenant | 계약 / 언약 | Depends on context. |
```

## Classical Chinese

For 한문/한자 texts:

- Keep original punctuation if the source has scholarly punctuation.
- If punctuation is absent, mark supplied punctuation as editorial.
- Provide literal translation first when needed, then natural Korean.
- Preserve line, chapter, fascicle, or juan numbering.
- Record variant characters and edition notes.
